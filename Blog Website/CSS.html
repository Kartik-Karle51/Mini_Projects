CSS (Cascading Style Sheets)

CSS is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout of web pages, colors, fonts, and other visual aspects.

With CSS, developers can separate content from design. This allows for more flexibility in web design and easier maintenance.

  • CSS Syntax:
    • selector { property: value; } - The basic structure of CSS.
  • CSS Selectors:

    • .class - Selects elements with a specified class.

    • #id - Selects an element with a specified id.

    • element - Selects all elements of a specified type (e.g., <div>).

  • CSS Properties:

    • color - Specifies text color.

    • font-size - Specifies the font size of text.

    • background-color - Specifies the background color of an element.

Example of a simple CSS rule: 

h1 {
    color: blue;
    font-size: 30px;
}
