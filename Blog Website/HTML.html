HTML (HyperText Markup Language)

HTML is the standard markup language used to create web pages. It is the skeleton of a webpage, used to structure content such as text, images, links, and multimedia.

HTML defines the meaning and structure of web content. It uses a system of tags and attributes to structure a document and embed content like headings, paragraphs, images, and tables.

  • Basic HTML Structure:
    • <html>: Defines the root of the HTML document.

    • <head>: Contains metadata and links to stylesheets, fonts, etc.

    • <body>: Contains the content visible to users.

  • Common HTML Tags:

    • <h1></h1> - Main Heading

    • <p></p> - Paragraph

    • <a></a> - Hyperlink

    • <img> - Image
